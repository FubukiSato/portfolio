〜product〜

- 個人開発
タイトル: Aiaru
概要: 会津大生用アルバイト掲示板です。 会津大生を募集しているお店、企業の方にアルバイトを登録していただきます。

- 開発背景
友人がアルバイト検索アプリでバイトを探している時に「ここのお店いいんだけど、自分の家から遠すぎるなぁ」, 「自分は学生なんだから、パートタイムでの募集は載せて欲しくない」と言っているのを聞いたからです。これらは自分も以前から思っていたことだったので、 これを機に学生限定で、かつ自分の住んでいる地域内でアルバイト募集をしている店しか載っていないwebアプリは需要があると思い作成しました。

開発期間: 2ヶ月半ほど
開発人数: 1人
開発言語: フロントエンド → HTML, CSS, jquery バックエンド → Ruby フレームワーク → Ruby on Rails
役割: 使用検討、設計、コーディング、テスト
URL: https://www.ai-aru.com
開発秘話: 実は、私は美術的センスが皆無で、フロントエンドのデザインは毎回、彼女に確認してもらっていました笑
